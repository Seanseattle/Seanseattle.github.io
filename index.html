打造一个程序员的Ubuntu系统

Posted on 2018-04-18
Words count in article: 666 字 | Reading time ≈ 3 分钟

 最近又新安装了一遍ubuntu，然后要把之前的软件安装一遍。发现一个一个去google还是有些麻烦的,而且有些软件可能都忘记安装。所以这里先整理一个列表，里面链了很多的教程,来安装这些程序猿们常用的软件。目前我需要的几个软件，Chrome，搜狗，Idea，Pycharm，wps，有道，vscode，postman，mysql，sublime，jdk，tomcat，ss…
借用了一下大佬的教程，罗列如下。

（说明，如果出现缺少依赖，先安装依赖，或者执行 sudo apt-get install -f 试试）

1.首先自然是安装一个输入法啊——搜狗输入法

https://www.jianshu.com/p/620f123da7cb

2.开始安装Chrome吧

sudo wget http://www.linuxidc.com/files/repo/google-chrome.list -P /etc/apt/sources.list.d/
wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
sudo apt-get update
sudo apt-get install google-chrome-stable

3.接下来安装idea，pycharm，这几个都是jetbrain公司的，真心喜欢他们公司的产品，用起来十分舒服。下面是官网，里面很多产品哦,解压到你想要的位置，然后运行该文件bin目录下的某个启动脚本就好了，比如idea是 ./idea.sh

https://www.jetbrains.com/

4.金山开发了linux系统下的wps，系统自带的Libre Office真心难用。软件下载之后输入

sudo dpkg -i bababa  #bababa就是你下载的deb包
#打开之后会提示你缺少部分自体，可以用一下方法修复
https://blog.csdn.net/liweibin1994/article/details/73384790

5.mysql的安装很简单的

sudo apt-get install mysql-server

然后自己设置一下root的密码就可以了。数据库软件直接用idea就好了

6.postman,vscode,sublime的安装起来都比较简单了。但是sublime不能输入中文，可以通过下面链接解决，但是目前个人还是喜欢用vscode了。

#解决sublime输入中文问题
https://www.jianshu.com/p/bf05fb3a4709
#vscode，下载之后，使用 sudo dpkg -i .... 
https://code.visualstudio.com/
#postman
https://www.getpostman.com/apps

7.jdk可以去官网下载，下面是安装和设置环境变量的教程

https://www.jianshu.com/p/8712cf20a5c9

8.下了chrome，当然要fanqiang找资源，嘿嘿。之前就已经搭好了ss服务器，所以本地只需要安装ss-client，然后写一个启动的脚本就好了。然后去chrome下载一个SwitchyOmega,配置一下就好了，可以参考以下教程

https://www.sundabao.com/ubuntu%E4%BD%BF%E7%94%A8shadowsocks/

9.当然我们也需要美化一下我们的Ubuntu系统啦，可以参照这个

https://tofar.github.io/2017/Ubuntu/Ubuntu%E7%BE%8E%E5%8C%96%E7%AC%94%E8%AE%B0/
